# FastApi Test Project
## Установка
* Клонируйте и скачайте репозиторий.
* Создайте виртуальное окружение и активируйте его.
* Введите в терминале "pip install -r requirements.txt" и запустите.
* Введите в терминале "python src/main.py" и запустите.
* Для просмотра эндпоинтов откройте страницу http://"host"/docs#.
## Настройка
* По умолчанию host = "127.0.0.1", но его можно изменить в файле config.py, что находится в папке src.
* Также в файле config.py можно изменить флаги для записей логов в файлы из папки logs.
* Уровень ведения логов можно изменить в файле logs.py из папки src, изменив значение глобальной переменной log_level на 75 строке этого файла.
## Эндпоинты
### GET http://"host"/incidents 
Получить все записи инцидентов.
### POST http://"host"/incidents 
Добавить запись инцидента.
### GET http://"host"/incidents/"status" 
Найти записи инцидентов по статусу.
### PUT http://"host"/"id"/"status"
Изменить статус инцидента по его id.
